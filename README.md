 Amazon Clone Website

 Overview

This is my first front-end web development project, an Amazon Clone website created using HTML and CSS. The project replicates the basic layout and design of Amazon's homepage to practice web development fundamentals.

 Features

* Responsive navigation bar
* Search bar design
* Hero section
* Product categories section
* Footer section
* Clean and organized HTML & CSS structure

Technologies Used

* HTML5
* CSS3

Purpose

The main goal of this project was to improve my skills in:

* HTML page structure
* CSS styling and layouts
* Flexbox
* Responsive web design basics
* Front-end development practices

 Future Improvements

* Add JavaScript functionality
* Implement product pages
* Create shopping cart features
* Improve responsiveness for all devices
